* Automated with **Ansible**.
* Packages installed with **Homebrew**.
* Dotfiles managed with **GNU Stow**.

```shell
git clone https://github.com/robertstojs/dotfiles.git
cd dotfiles
make
# or when only running a specific tag (defaults to 'all')
make stow
```
